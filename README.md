# Arsha Electric.co Next.js Website

Local SEO starter site for Arsha Electric.co.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Create a new GitHub repo.
2. Upload/push this folder.
3. In Vercel, click **Add New → Project**.
4. Import the GitHub repo.
5. Deploy.
6. Add your domain under **Project → Settings → Domains**.

## Before launch

- Replace placeholder phone/email in `app/data.ts`.
- Replace `business.url` with your final domain.
- Connect Google Search Console.
- Add the website link to your Google Business Profile.
- Add real project photos and testimonials.
